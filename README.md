# ServiceSathi - Empowering Local Service Providers
 🚀 About ServiceSathi
ServiceSathi is a **full-stack platform** designed to bridge the gap between **local service providers** and customers, ensuring **fair wages** and **direct connections** without any middlemen. Our mission is to empower skilled professionals while making it easier for users to find trusted local services.

---

 🌟 Key Features
- **No Middleman** – Directly connect with service providers, ensuring **transparent pricing** and **fair wages**.
- **Support Local Businesses** – Strengthen the local economy by hiring nearby skilled professionals.
- **Secure & Reliable** – Verified service providers and **real customer reviews**.
- **Easy Booking** – Hassle-free scheduling with real-time availability.
- **Fair Wage Model** – Service providers receive their full earnings without unfair commission cuts.
- **User-Friendly Interface** – Simple, intuitive design for a seamless experience.

---

🏗️ Tech Stack
### Frontend:
- React.js (with Tailwind CSS for UI design)
- Next.js for server-side rendering (SSR) and SEO
- Framer Motion for animations

### Backend:
- Node.js & Express.js
- MongoDB (for scalable database management)
- Firebase Authentication (for secure login/signup)

### Deployment:
- Vercel (Frontend)
- Azure(Backend & Database Hosting)

## 🛠️ Installation & Setup
### Prerequisites:
- **Node.js** (v16+ recommended)
- **MongoDB** (local or cloud-based)
- **Git**

### Steps:
1. Clone the repository:
   ```sh
   git clone https://github.com/Aditya0825-crypto/servicesathi.git
   cd servicesathi
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Set up environment variables:
   - Create a `.env` file in the root directory and add:
     ```sh
     NEXT_PUBLIC_APP_URL=<your_app_url>
     MONGODB_URI=<your_mongodb_connection_string>
     FIREBASE_API_KEY=<your_firebase_api_key>
     ```
4. Start the development server:
   ```sh
   npm run dev
   ```
5. Open `http://localhost:3000` in your browser.
